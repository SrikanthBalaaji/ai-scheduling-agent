# AI Scheduling Agent

## Project Overview

AI-powered student planner with:

* Event Billboard
* Personal Calendar
* AI Agent for scheduling decisions

## Tech Stack

* Backend: FastAPI
* Frontend: React (Vite)
* Database: SQLite
* Agent: LangGraph (coming later)
